# シンガポール（SGX）個別株リサーチガイド

## 1. セクター概要

シンガポール証券取引所（SGX）はアジアの金融ハブとして、東南アジア最大級の株式市場を運営している。Straits Times Index（STI）は30銘柄で構成され、時価総額の約50%を3大銀行（DBS, OCBC, UOB）が占める。

### 市場の主要特性

| 特性 | 内容 |
|------|------|
| 取引所 | Singapore Exchange（SGX） |
| 主要指数 | Straits Times Index（STI）、FTSE ST All-Share Index |
| 取引通貨 | SGD（シンガポールドル） |
| 取引時間 | 9:00-12:00, 13:00-17:00（SGT / UTC+8） |
| 決済 | T+2 |
| ティッカー形式 | `.SI`（例: `D05.SI`） |
| 開示言語 | 英語（標準） |
| 会計基準 | SFRS(I) = IFRSと同等 |
| REIT数 | 約40銘柄、時価総額S$1,000億超 |

### セクター別構成

| セクター | 特徴 | STI構成比（概算） |
|---------|------|-----------------|
| 銀行・金融 | 3大銀行がSTIの約50%を占める | 50%超 |
| REIT | 約40銘柄、アジア最大のREIT市場の一つ | 15-20% |
| 通信 | Singtel中心。2025年にM1→Simba統合 | 5-8% |
| 不動産 | CapitaLand系、CDL、UOL等 | 8-10% |
| 海運・産業 | Keppel, Sembcorp, Yangzijiang等 | 8-10% |
| テクノロジー | SGX上場は限定的。Sea/GrabはNYSE/NASDAQ上場 | 3-5% |

## 4. セクター固有KPIと取得方法

### 4.1 銀行セクター

| KPI | 定義 | データソース | 重要度 |
|-----|------|------------|--------|
| **NIM**（Net Interest Margin） | 純利鞘。貸出金利と調達金利の差。2024年平均約2.1% | 各行四半期決算 | ★★★★★ |
| **NPL Ratio**（不良債権比率） | 不良債権/総貸出。DBS 1.1%, OCBC 0.9%, UOB 1.5%（2024年末） | 各行四半期決算 | ★★★★★ |
| **CET1 Ratio** | Common Equity Tier 1自己資本比率。3行平均15.3%（2024年末） | 各行四半期決算、MAS | ★★★★★ |
| **ROE** | 自己資本利益率 | 各行四半期決算 | ★★★★★ |
| **Fee Income** | 非金利収入（ウェルスマネジメント、投資銀行、トレーディング） | 各行四半期決算 | ★★★★☆ |
| **Wealth Management AUM** | ウェルスマネジメント運用資産残高 | 各行四半期決算 | ★★★★☆ |
| **Cost-to-Income Ratio** | 経費率 | 各行四半期決算 | ★★★★☆ |
| **Loan Growth** | 貸出残高成長率 | 各行四半期決算 | ★★★★☆ |
| **CASA Ratio** | 当座・普通預金比率（調達コスト指標） | 各行四半期決算 | ★★★☆☆ |
| **Dividend Payout Ratio** | 配当性向 | 各行決算、MAS配当ガイダンス | ★★★★☆ |

### 4.2 S-REIT セクター

| KPI | 定義 | データソース | 重要度 |
|-----|------|------------|--------|
| **DPU**（Distribution Per Unit） | ユニットあたり分配金 | 各REIT半期/四半期決算 | ★★★★★ |
| **Distribution Yield** | 分配利回り = DPU / ユニット価格 | 各REIT決算、市場データ | ★★★★★ |
| **NAV per Unit** | ユニットあたり純資産価値 | 各REIT決算（年1回鑑定評価） | ★★★★★ |
| **Gearing Ratio** | 総借入 / 総資産。上限50%（MAS規制） | 各REIT四半期決算 | ★★★★★ |
| **ICR**（Interest Coverage Ratio） | EBITDA / 支払利息。最低1.5倍（MAS規制） | 各REIT四半期決算 | ★★★★★ |
| **WALE**（Weighted Average Lease Expiry） | 加重平均リース残存期間（年） | 各REIT四半期決算 | ★★★★☆ |
| **Occupancy Rate** | 稼働率 | 各REIT四半期決算 | ★★★★★ |
| **NPI**（Net Property Income） | 純不動産収入 | 各REIT四半期決算 | ★★★★★ |
| **NPI Yield** | NPI / 資産評価額 | 各REIT決算 | ★★★★☆ |
| **Weighted Average Cost of Debt** | 加重平均借入コスト | 各REIT四半期決算 | ★★★★☆ |
| **Fixed Rate Debt %** | 固定金利借入比率 | 各REIT四半期決算 | ★★★★☆ |
| **Sponsor Pipeline** | スポンサーからの取得可能資産パイプライン | 各REIT投資家プレゼン | ★★★★☆ |
| **Rental Reversion** | リース更新時の賃料改定率 | 各REIT四半期決算 | ★★★★☆ |
| **P/NAV** | ユニット価格 / NAV per unit（プレミアム/ディスカウント） | 市場データ | ★★★★☆ |

### 4.3 不動産セクター

## 6. シンガポール固有の注意点

### 6.1 S-REIT制度の詳細

S-REITはシンガポール市場の最大の差別化要因であり、約40銘柄、時価総額S$1,000億超のアジア有数のREIT市場を形成している。

#### 税制優遇

| 項目 | 内容 |
|------|------|
| **税透過性（Tax Transparency）** | 課税所得の90%以上を分配すれば、REIT法人レベルでは非課税 |
| **個人投資家（税務居住者）** | 分配金は非課税 |
| **個人投資家（非居住者）** | 分配金に10%源泉徴収（2025年12月末まで軽減措置、延長の可能性あり） |
| **法人投資家** | 通常の法人税率（17%）で課税 |
| **不動産取得税** | REIT向けの優遇税率あり |

#### レバレッジ規制（MAS）

| 規制項目 | 内容 |
|---------|------|
| **Aggregate Leverage Limit** | 総資産の**50%**が上限（2020年にCOVID対応で45%→50%に引上げ） |
| **ICR（Interest Coverage Ratio）** | 最低**1.5倍**（2024年にMASが規制統一。従来はレバレッジ45%超の場合のみICR 2.5倍要件） |
| **追加開示要件** | 2025年3月よりレバレッジ・ICRに関する追加開示義務 |

#### スポンサー制度

| 項目 | 内容 |
|------|------|
| **スポンサーの役割** | 資産パイプライン提供、経営支援、ブランド信用力 |
| **スポンサーパイプライン** | スポンサーが保有する未注入資産。REIT成長の重要ドライバー |
| **利益相反管理** | MASとSGX規則で利益相反取引に対する独立取締役承認・ユニットホルダー投票要件 |
| **主要スポンサー** | CapitaLand Investment（7 REIT）、Mapletree Investments（3 REIT）、Frasers Property（2 REIT）、Keppel（3 REIT） |

#### S-REIT分析フレームワーク

```
□ 収益安定性
  - Occupancy Rate（稼働率 > 90%が目安）
  - WALE（3年以上が望ましい）
  - テナント集中度（トップ10テナントの占有率）
  - Rental Reversion（正のリバージョンが望ましい）

□ 財務健全性
  - Gearing Ratio（< 40%が保守的、> 45%は注意）
  - ICR（> 2.5倍が望ましい）
  - 固定金利比率（> 70%が理想的な金利リスクヘッジ）
  - Weighted Average Debt Maturity（3年以上が望ましい）

□ 成長ポテンシャル
  - スポンサーパイプラインの規模
  - AEI（Asset Enhancement Initiative）計画
  - 海外展開の多様化
  - 有機的成長（Built-in escalation等）

□ バリュエーション
  - Distribution Yield vs 10年SGS利回り（スプレッドを確認）
  - P/NAV（プレミアム/ディスカウント）
  - 同セクターREITとのYield比較
```

### 6.2 政府系企業（GLC）の影響力

シンガポール経済において、政府系投資会社Temasek HoldingsとGIC（Government of Singapore Investment Corporation）は極めて大きな影響力を持つ。

| 投資会社 | 運用資産 | 役割 | 主要保有企業（SGX上場） |
|---------|---------|------|---------------------|
| **Temasek Holdings** | US$2,880億超（2024年） | 政府系投資会社。シンガポール国内外に投資。SGX時価総額の約25%を占める | DBS（29%）, Singtel（52%）, Keppel, Sembcorp, ST Engineering, Mapletree系 |
| **GIC** | US$7,440億超（2022年推定） | ソブリン・ウェルス・ファンド。外貨準備の運用。主にグローバル分散投資 | 直接的なSGX上場銘柄の大量保有は限定的（非開示方針） |

#### GLCの特徴

- Temasek系GLCは**SGX時価総額の約25%**を占める
- GLCは非GLC企業と比べて**バリュエーションプレミアム**が認められる傾向（政府の信用力、ガバナンス水準）
- Temasekは商業ベースで運用しており、政府の直接介入は限定的
- GLCは概してコーポレートガバナンスが良好で、独立取締役比率が高い

### 6.3 外国人投資制限

シンガポールは基本的に**オープンな投資環境**だが、一部セクターに制限がある。

| セクター | 制限内容 |
|---------|---------|
| **銀行** | Banking Act上、MASの承認なく銀行株の5%超取得不可。12%超は大株主（Substantial Shareholder）要件 |
| **メディア** | Newspaper and Printing Presses Act上、メディア企業の外国人持分に制限 |
| **不動産（住居用）** | 外国人のHDB（公営住宅）購入不可。ABSD（追加買主印紙税）が課される |
| **その他上場株** | 原則として制限なし。自由に取引可能 |

### 6.4 不動産クーリングメジャー（ABSD等）

シンガポール政府は不動産市場の過熱を抑制するため、段階的にクーリングメジャーを導入している。

#### ABSD（Additional Buyer's Stamp Duty）

| 買主区分 | 1件目 | 2件目 | 3件目以降 |
|---------|------|------|---------|
| シンガポール市民 | 0% | 20% | 30% |
| 永住権保有者（SPR） | 5% | 30% | 35% |
| **外国人** | **60%** | 60% | 60% |
| 法人（事業体） | 65% | 65% | 65% |

> **注**: 2023年4月の改定で外国人のABSDが30%→60%に倍増。不動産セクターに大きな影響を与えた。

#### その他の規制

| 規制 | 内容 |
|------|------|
| **TDSR**（Total Debt Servicing Ratio） | 借入者の月間債務返済額が月収の55%を超えてはならない |
| **LTV**（Loan-to-Value） | 初回住宅ローン: 75%（HDB）/ 75%（銀行）。2件目以降は段階的に引下げ |
| **SSD**（Seller's Stamp Duty） | 購入後3年以内の売却に課税（1年目12%→2年目8%→3年目4%） |

### 6.5 SGDの安定性とMASの金融政策

MASは世界でもユニークな**為替レートベースの金融政策**を採用している。

| 項目 | 内容 |
|------|------|
| **政策ツール** | S$NEER（名目実効為替レート）のバンド管理。金利ではなく為替で物価安定を図る |
| **BBCフレームワーク** | Basket（通貨バスケット）、Band（変動許容幅）、Crawl（緩やかな増価トレンド） |
| **政策レビュー** | 年2回（通常4月・10月）。傾き・幅・中心レベルの3パラメータを調整 |
| **背景** | GDP比300%超の貿易依存度。金利よりも為替レートがインフレに強い影響を持つ |
| **SGDの特性** | 長期的に緩やかな増価トレンド。対USD、対主要通貨で安定的 |

> **投資への含意**: SGDは相対的に安定した通貨であり、為替リスクは他の新興国通貨より低い。ただし、MASの政策バンド変更（引締め/緩和）は銀行のNIM、REIT の外貨建資産評価に影響する。

### 6.6 会計基準

| 項目 | 内容 |
|------|------|
| **基準名** | SFRS(I)（Singapore Financial Reporting Standards - International） |
| **IFRSとの互換性** | 実質的に同等。IASB発行のIFRSに準拠 |
| **デュアルリスティング** | SGXと海外取引所の二重上場企業はIFRS使用も可 |
| **銀行固有** | MASが銀行向けに貸倒引当金の修正要件（Notice to Banks）を発行 |
| **監督機関** | ACRA（会計基準の設定）、ISCA（シンガポール勅許会計士協会） |

### 6.7 デュアルリスティング

一部のシンガポール企業はSGXと海外取引所に二重上場している。また、海外企業のSGXセカンダリー上場もある。

| パターン | 例 |
|---------|-----|
| SGX + NYSE/NASDAQ | Sea Limited (SE)、Grab (GRAB) はNYSE/NASDAQ のみ上場 |
| SGX + HKEX | Jardine Matheson系企業（歴史的にSGX上場、一部HKEX移行） |
| SGX + ASX | 一部資源・不動産企業 |
| 海外企業のSGX上場 | Yangzijiang（中国造船）等 |

## 7. 分析チェックリスト

### 7.1 シンガポール株 共通チェックリスト

```
□ 企業概要
  - 主要事業・収益源
  - GLC（政府系）か否か。Temasek/GIC持分の確認
  - 海外売上比率（シンガポール以外のエクスポージャー）

□ 財務分析
  - 過去5年の売上・利益トレンド
  - ROE、ROA（銀行はROE特に重要）
  - 配当政策と配当利回り
  - キャッシュフロー推移

□ バリュエーション
  - P/E、P/B、EV/EBITDA（セクターに応じて）
  - 同業比較（STI構成銘柄内・ASEAN同業）
  - 配当利回り vs SGS（シンガポール国債）利回り

□ ガバナンス
  - 独立取締役比率
  - 関連当事者取引の有無・規模
  - 経営陣の株式保有（Skin in the game）
  - SGXコーポレートガバナンスコード準拠状況

□ リスク要因
  - SGD為替リスク（外貨建資産・収益がある場合）
  - 金利環境（MAS政策、米金利との連動）
  - 地政学リスク（米中関係、ASEAN地域リスク）
  - 規制変更リスク（不動産クーリングメジャー、銀行規制等）
```

### 7.2 S-REIT 詳細分析チェックリスト

```
□ ポートフォリオ品質
  - 資産タイプ（オフィス/リテール/産業/DC/ヘルスケア等）
  - 地理的分散（シンガポール比率、海外資産の通貨リスク）
  - 資産年齢と物理的状態
  - テナントプロファイル（業種分散、信用力、トップテナント集中度）

□ 収益安定性
  - Occupancy Rate（過去4四半期トレンド）
  - WALE（By NLA and By Gross Revenue）
  - Rental Reversion（正/負、セクターサイクル考慮）
  - Built-in Rent Escalation（固定引上げ条項の有無）

□ 財務健全性
  - Gearing Ratio（< 40%: 保守的、40-45%: 適度、> 45%: 注意）
  - ICR（> 2.5x: 良好、1.5-2.5x: 注意、< 1.5x: 危険）
  - All-in Financing Cost（加重平均借入コスト）
  - Fixed Rate Debt比率（> 70%: 良好な金利ヘッジ）
  - Weighted Average Debt Maturity（> 3年: 良好）
  - Debt Maturity Profile（満期集中リスクの有無）

□ 成長戦略
  - スポンサーパイプライン（規模、資産タイプ、注入タイムライン）
  - Third-party Acquisition実績
  - AEI（Asset Enhancement Initiative）計画と過去のROI
  - 開発プロジェクト（BTS = Build-to-Suit等）

□ スポンサー評価
  - スポンサーの財務力・信用格付
  - スポンサー持分比率（コミットメントの指標）
  - スポンサーとの利益相反リスク
  - スポンサーの不動産パイプラインの深さ

□ バリュエーション
  - Distribution Yield vs セクター平均
  - Distribution Yield vs 10年SGS利回り（スプレッド）
  - P/NAV（プレミアム: > 1.0x、ディスカウント: < 1.0x）
  - DPU成長率トレンド
  - Yield Spread（他REITとの比較）

□ 金利感応度
  - MASの金融政策方向性
  - 変動金利借入の比率
  - 金利1%上昇時のDPUへの影響（Sensitivity Analysis）
```

### 7.3 銀行セクター分析チェックリスト

```
□ 収益構造
  - NIM（Net Interest Margin）トレンドと方向性
  - Fee Income構成（Wealth Management, Trade Finance, Cards等）
  - Trading Income（持続性の評価）
  - Non-Interest Income / Total Income比率

□ 資産品質
  - NPL Ratio（不良債権比率）トレンド
  - SP（Specific Provisions）/ 一般引当金の推移
  - NPL Coverage Ratio（引当金カバー率 > 100%が望ましい）
  - セクター別・地域別のNPL内訳

□ 資本・自己資本
  - CET1 Ratio（> 13%: 良好）
  - 配当政策（Payout Ratio、特別配当の有無）
  - 自社株買い計画（2025年はDBS, OCBC, UOBともに積極的）
  - MASの配当ガイダンス

□ 地域展開
  - ASEAN展開（UOBはCitiリテール買収後のASEAN統合進捗）
  - 中国・香港エクスポージャー（不動産リスク）
  - インドネシア・タイ・マレーシア等の成長市場

□ 競争環境
  - デジタルバンク（GrabとSeaがデジタルバンクライセンス取得済）
  - Wealth Management競争（香港との比較）
  - フィンテックの影響
```
