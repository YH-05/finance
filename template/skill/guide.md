## ツール活用ガイド

<!-- このスキルで使用するツールの詳細な使い方 -->

### ファイル操作

#### Read ツール

**用途**: ファイルの読み取り

```
Read(file_path="/path/to/file.md")
```

**使用シーン**:
- 既存ファイルの内容確認
- テンプレートの読み込み

#### Write ツール

**用途**: ファイルの作成・上書き

```
Write(file_path="/path/to/file.md", content="...")
```

**注意点**:
- 既存ファイルは上書きされる
- ディレクトリが存在しない場合は作成される

#### Edit ツール

**用途**: ファイルの部分編集

```
Edit(
  file_path="/path/to/file.md",
  old_string="置換前の文字列",
  new_string="置換後の文字列"
)
```

**使用シーン**:
- マーカーセクションの更新
- 特定部分の修正

### 検索ツール

#### Glob ツール

**用途**: パターンによるファイル検索

```
Glob(pattern="**/*.md")
```

**使用シーン**:
- 特定種類のファイル一覧取得
- ディレクトリ内のファイル数確認

#### Grep ツール

**用途**: ファイル内容の正規表現検索

```
Grep(pattern="class.*Error", path="src/")
```

**使用シーン**:
- コード内のパターン検索
- 特定の文字列を含むファイル特定

### 外部ツール（Bash 経由）

#### gh CLI

**用途**: GitHub 操作

```bash
# Issue 一覧
gh issue list --json number,title,state

# PR 作成
gh pr create --title "タイトル" --body "本文"

# Project 操作
gh project item-list 18 --owner @me --format json
```

#### MCP ファイルシステム

**用途**: ディレクトリ構造取得

```
mcp__filesystem__directory_tree(
  path=".",
  excludePatterns=["node_modules", ".git", "__pycache__"]
)
```
