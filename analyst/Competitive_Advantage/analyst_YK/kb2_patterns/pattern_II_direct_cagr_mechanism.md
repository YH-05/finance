# 高評価パターンII: 直接的なCAGR接続メカニズム

## 種別: 高評価パターン
## 確信度: 90%

## パターン定義

優位性→CAGRパラメータへの因果チェーンが短く（1-2ステップ）、各ステップが開示データで検証可能な場合に最高評価となる。

## 具体例

### COST顧客粘着力→CAGR（90%）
- KY: 「上級会員ミックス改善結果としてのARPU向上は既存店売上成長に直接影響ある項目である。また、上級会員/通常会員比率、上級会員の売上/売上寄与が適宜開示されており、貢献度合いも想定可能であるため、CAGR接続への納得度高い」
- **因果チェーン**: 会員更新率維持 → 上級会員ミックス改善 → ARPU向上 → 既存店売上成長（2ステップ、開示データで検証可能）

### COST国際展開→CAGR（90%）
- KY: 「新規出店への接続は妥当。できればCAGRのうちの新規出店寄与2-2.5%への確度は大きく上昇する」
- **因果チェーン**: 現地適応（tailor）→ 米国優位性の横展開 → 新規出店+2-2.5%（直接的）

### CHD#6→CAGR（90%）
- KY: 「日用品製造にあってマーケティングや新製品は売り上げの主要ドライバーとなる」
- **因果チェーン**: マーケティング/新製品 → 売上の主要ドライバー → 新商品寄与+1-1.5%（業界特性に合致）

## KYのルール

> 「貢献度合いも想定可能であるため、CAGR接続への納得度高い」

## 高評価の条件

| 条件 | 説明 |
|------|------|
| 因果チェーンが短い | 1-2ステップ |
| 開示データで検証可能 | 定期開示される比率・金額で追跡可能 |
| 業界特性に合致 | その業界で主要ドライバーであること |
| CAGRパラメータとの直接接続 | 既存店売上、新規出店、マージン等に直結 |

## 低評価になるCAGR接続

| パターン | 確信度 | 例 |
|---------|--------|-----|
| 増収前提のレバレッジ接続 | 20-50% | 退職率低下→営業レバレッジ+0.25% |
| 数値の根拠不明 | 30-50% | 売上成長+1-2%の背景不明 |
| 既存店売上への抽象的接続 | 50% | 低価格→既存店効果 |
| TAM/シェアの中間ステップ欠如 | 30-50% | 優位性→直接CAGR数値 |
