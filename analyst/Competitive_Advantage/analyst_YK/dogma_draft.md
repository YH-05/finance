## 1. 競争優位性の基本哲学

### 1.1 優位性とは何か

| # | 原則 | 検証 |
|---|------|------|
| 1 | 優位性は「能力・仕組み」であり、「結果・実績」ではない | ○ / × / △ |
| 2 | 優位性は「名詞」で表現される属性である（ブランド力、スイッチングコスト、技術力等） | ○ / × / △ |
| 3 | 優位性は競合に対して「相対的に際立つ」ものでなければならない | ○ / × / △ |
| 4 | 優位性は業界構造・競争環境と密接に関係する（K の Dogma と同一見解） | ○ / × / △ |
| 5 | 優位性の評価には定量的裏付けが重要（数値がなくても却下はしないが、あると納得度が上がる） | ○ / × / △ |

**抽出根拠:**
- 原則1: CHD#4「国際事業成長加速は結果であって背景となる優位性ではない」、CHD#5「ポート管理運営力そのものが優位性であり、ポートシフトは結果」
- 原則2: LLY feedback「優位性は名詞で表現される属性であり、ブランド力、スイッチングコスト、エコシステム...」
- 原則3: LLY#6「グローバル大手医薬品であればほぼ同等に備わっているネットワーク」「競争優位性として機能するには、相対的に際立つ必要がある」
- 原則4: ORLY#2, #5（市場構造との合致で90%の最高評価）
- 原則5: COST#1「意味のある数値裏付けは納得感を高める印象」

### 1.2 優位性と認めないもの

| # | 却下基準 | 具体例 | 検証 |
|---|---------|--------|------|
| 1 | **結果の優位性への誤帰属**: 高シェア、成長実績、高収益は結果であり原因ではない | MNST#1「シェアはあくまで結果」、CHD#4「成長加速は結果」 | ○ / × / △ |
| 2 | **業界共通の能力**: メガファーマなら誰でも持っている、多くの企業が言っていること | LLY#6「グローバル医療機関ネットワーク=大手共通」、COST#5「多くの企業が言っていること」 | ○ / × / △ |
| 3 | **戦略の優位性への混同**: 戦略は意思決定であり、優位性はそれを可能にする構造的な力 | ORLY#2「ドミナント出店戦略=戦略は優位性ではないのでは」 | ○ / × / △ |
| 4 | **定性的で測定不能な主張**: 評価のしようがない曖昧な差別化 | COST#2「売り場づくりの定性的差異がわかりにくく定量評価もしがたい」 | ○ / × / △ |
| 5 | **因果関係の飛躍**: 3段階以上の推論チェーンで各段階の裏付けがないもの | MNST#5「低所得→コンビニ購買率高い→粘着力高い は飛躍」 | ○ / × / △ |
| 6 | **事実誤認に基づく仮説**: 財務指標の誤読等 | MNST#6「OPMをGPMと誤認」 | ○ / × / △ |

## 2. 優位性の評価体系

### 2.1 確信度スケール

| ランク | 確信度 | 定義 | 検証 |
|--------|--------|------|------|
| かなり納得 | 90% | 構造的優位性 + 明確なCAGR接続 + 定量的裏付け | ○ / × / △ |
| おおむね納得 | 70% | 妥当な仮説 + 一定の裏付け（競合比較または数値あり） | ○ / × / △ |
| まあ納得 | 50% | 方向性は認めるが裏付け不十分（追加情報があれば上がりうる） | ○ / × / △ |
| あまり納得しない | 30% | 飛躍的解釈・因果関係の逆転・差別化根拠不十分 | ○ / × / △ |
| 却下 | 10% | 事実誤認・競争優位性として不成立 | ○ / × / △ |

### 2.2 評価を上げる要因（納得度を高めるもの）

| # | 要因 | 効果 | 具体例 | 検証 |
|---|------|------|--------|------|

| 3 | ネガティブケース（断念例、失敗例）による裏付け | +10-20% | CHD#7: バリュエーションによる買収断念例 → 50% | ○ / × / △ |
| 4 | 開示データによる検証可能性 | +20%程度 | COST顧客粘着力: 会員比率の開示 → 90% | ○ / × / △ |
| 5 | 業界構造との合致 | +30%程度 | ORLY#2: フラグメント市場 × 規模・密度 → 90% | ○ / × / △ |
| 6 | 能力・仕組みの具体的説明 | +10-20% | CHD#1: 買収基準の明確化、人員投入プロセス → 70% | ○ / × / △ |

### 2.3 評価を下げる要因（納得度を低下させるもの）

## 6. K の Dogma との比較

| 観点 | K の Dogma | Y の判断軸（本ドラフト） | 共通点/差異 |
|------|-----------|------------------------|-------------|
| 競争優位性の定義 | 長い年月をかけて形成される構造的な事業特性 | 能力・仕組みであり結果ではない。名詞で表現される属性 | **共通**: 構造的・持続的。**差異**: Y はより操作的な定義（名詞、能力vs結果） |
| 業界構造の重視 | 最上位に配置（独占/複占/寡占/フラグメント等） | 同様に重視（ORLY 90%の根拠） | **共通**: 業界構造を前提条件として重視 |
| 高シェアの扱い | 「何かしらの競争優位性の結果としてもたらされるもの」 | 「シェアはあくまで結果」 | **完全一致** |
| 差別化製品の扱い | 「差別化自体が目的化されたものには留意」 | 直接言及なし（ただし結果vs原因の判断で同旨） | **方向性一致** |
| 優秀な経営陣 | 「経営陣の優劣によって左右される事業にそもそも持続的な競争優位性があるかが疑わしい」 | 直接言及なし | K のみ明示的 |
| ビジネスモデル分類 | 13類型を列挙 | 直接言及なし（ただし個別銘柄で暗黙的に適用） | K がより体系的 |
| CAGR接続 | 言及なし | 独立した2軸評価（優位性+CAGR接続） | Y のみ明示的 |
| 定量的裏付け | 間接的に言及 | 明示的に重視（「数値裏付けは納得感を高める」） | Y がより強調 |
| 警鐘機能 | 言及なし | 最も重要な期待機能 | Y のみ |
