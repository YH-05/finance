# ルール4: 定量的裏付けがあると納得度が上がる

## カテゴリ: 裏付けの質

## ルール定義

具体的な数値（退職率、報酬比率、対競合比較データ、シェア、成長率等）による定量的裏付けがある主張は納得度が上がる。逆に、定性的で測定不能な主張は評価しがたい。

## 評価への影響

| 裏付けの種類 | 効果 |
|-------------|------|
| 定量的裏付け（数値、比率、対競合データ） | +20%程度 |
| 特定競合との具体的比較 | +10-20% |
| 開示データによる検証可能性 | +20%程度 |
| 定量データ皆無 | -10-20% |

## 良い例（KYが高評価したケース）

### COST#1: 退職率9% vs 業界20%+（70%）
- 「意味のある数値裏付けは納得感を高める印象」
- **ポイント**: 業界平均との具体的な数値比較が納得度を高めた

### CHD#2: 粗利額25%ボーナス制度（70%）
- 「粗利額の一定割合25%をボーナス原資とする組織的な報酬の考えは売り上げ増への動機づけとして有効」
- **ポイント**: 具体的な報酬制度の数値が説得力を持った

### LLY#2: Novo対比での経口GLP-1差別化（70%）
- 「競合であるNovoの経口薬剤と比較し、食事等の制限や製造面での優位性が確認されており」
- **ポイント**: 具体的な競合名と比較軸が明確

## 悪い例（KYが低評価したケース）

### COST#2: 劇場的価値（10% — 却下）
- 「売り場づくりの定性的差異がわかりにくく、改善寄与などの定量的評価もしがたい」
- **ポイント**: 数値も比較も一切なく、定性的な主張のみ

### CHD#6: デジタル・マーケティング能力（50%）
- 「根源となる要因や他社対比の優位性についての言及なく納得感乏しい」

### CHD#7: 規律ある意思決定（50%）
- 「言葉でしか説明されない傾向が多い」

## KYの原則

> 「意味のある数値裏付けは納得感を高める印象」（COST#1 feedback）

定量的裏付けとして有効なもの:
1. 業界平均との比較数値
2. 特定競合との比較データ
3. 時系列での変化率
4. 開示データ（検証可能な数値）
