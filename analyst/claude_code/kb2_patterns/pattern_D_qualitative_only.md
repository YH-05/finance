# 却下パターンD: 定性的で定量的裏付けなし

## 種別: 却下パターン
## 確信度: 10-30%

## パターン定義

数値も比較データもなく、定性的な表現のみで優位性を主張するパターン。「〜が優れている」「〜が差別化されている」と述べるだけで、測定可能な裏付けがない。

## 具体例

### COST#2: 劇場的価値による差別化（10% — 却下）
- KY: 「売り場づくりの定性的差異がわかりにくく、改善寄与などの定量的評価もしがたい」
- **問題**: 「劇場的価値」「エンターテイメント」という概念的な主張のみ。定量データ皆無

### CHD#6: デジタル・マーケティング能力（50%）
- KY: 「根源となる要因や他社対比の優位性についての言及なく納得感乏しい」
- **問題**: マーケティング能力を主張するが、競合比較データがない

### CHD#7: 意思決定の迅速さ、規律（50%）
- KY: 「言葉でしか説明されない傾向が多い」
- **問題**: 「迅速」「規律ある」という定性的表現のみ

## KYのルール

> 「定量的裏付け（数値、比率、対競合比較）がなければ『優位性として評価しがたい』」

## 定量的裏付けとして有効なもの

| 裏付けの種類 | 例 | 効果 |
|-------------|-----|------|
| 業界平均との比較 | 退職率9% vs 業界20%+ | 強い |
| 競合との直接比較 | AZO対比での補充頻度 | 強い |
| 時系列変化 | シェア推移、利益率推移 | 中程度 |
| 絶対数値 | PB比率50%超、会員更新率90% | 中程度 |

## 検出のチェックポイント

- 主張に数値が含まれているか？
- 競合との比較データがあるか？
- 「〜が優れている」の「優れている」を測定できるか？
- 業界データとの対比があるか？
