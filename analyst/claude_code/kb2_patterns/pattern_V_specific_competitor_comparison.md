# 高評価パターンV: 競合との具体的比較による差別化確認

## 種別: 高評価パターン
## 確信度: 70%

## パターン定義

競合名を特定し、具体的な差別化要因を示す仮説。「一般的に優れている」ではなく「Xと比べてYの点で優位」と示すことで納得度が上がる。

## 具体例

### ORLY#1: AZO対比でのDIFM補充頻度の優位性（70%）
- KY: 「主力競合の1社であるAZO対比では、部品補充頻度も高いなどORLYのDIFM事業の優位性に起因する粘着性の高さを評価」
- **ポイント**: 具体的な競合名（AZO）と具体的な差別化軸（補充頻度）

### MNST#4: REDBULLとの戦略差異（70%）
- KY: 「REDBULLは単一色の強いブランドであり...MNSTは競争上もREDBULLと違い、複数ブランドによる多様な消費層とりこむことで、結果としての数量成長につながる」
- **ポイント**: 最大の競合（REDBULL）との明確な戦略的差異（単一 vs 複数ブランド）

### COST#7: PBのkirklandブランド認知（70%）
- KY: 「kirklandなどのPBブランドの認知は進んでいる印象、NBベンダーへの価格交渉のツールにも生かせる」
- **ポイント**: 具体的なブランド名（kirkland）と競合に対する交渉力への接続

## KYのルール

> 競合名を特定し、具体的な差別化要因を示す仮説に高評価を与える。

## 効果的な競合比較の条件

| 条件 | 説明 |
|------|------|
| 具体的な競合名 | 「競合」ではなく「AZO」「REDBULL」等 |
| 比較軸の明確化 | 補充頻度、ブランド戦略、製造法等 |
| データによる裏付け | 可能であれば数値比較 |
| 純粋競合であること | 異業種ではなく同業態の直接的競合 |

## KYの業界知識による補完

KYは自身の業界知識で仮説を補強・批判することがある:
- COST#4: 「FEDEXやUPSなどの事業者のコメントなどからそのような考えに至った経緯有」
- MNST#5: 「REDBURN社のレポート24/8/13から、REDBULLはプレミアム、MNSTは廉価にセグメント分け」
