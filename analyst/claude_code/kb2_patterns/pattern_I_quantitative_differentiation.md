# 高評価パターンI: 定量的裏付けのある差別化

## 種別: 高評価パターン
## 確信度: 70-90%

## パターン定義

具体的な数値（退職率、報酬比率、対競合比較データ等）による定量的裏付けがあり、差別化が明確な主張。KYは「意味のある数値裏付けは納得感を高める印象」と述べている。

## 具体例

### COST#1: 退職率9% vs 業界平均20%+（70%）
- KY: 「意味のある数値裏付けは納得感を高める印象」
- **ポイント**: 業界平均との明確な数値差が、主張の説得力を高めた

### CHD#2: 粗利額25%ボーナス制度（70%）
- KY: 「粗利額の一定割合25%をボーナス原資とする組織的な報酬の考えは売り上げ増への動機づけとして有効であり、結果的に一人当たり売上高が高位となる」
- **ポイント**: 具体的な報酬制度の仕組みとその効果が数値で示された

### LLY#2: Novo対比での経口GLP-1の具体的差別化（70%）
- KY: 「競合であるNovoの経口薬剤と比較し、食事等の制限や製造面での優位性が確認されており、注射剤の場合よりも差別化がなされている」
- **ポイント**: 具体的な競合名と、具体的な差別化軸（食事制限、製造法）

## KYのルール

> 「意味のある数値裏付けは納得感を高める印象」

## 定量的裏付けの種類と効果

| 種類 | 効果 | 例 |
|------|------|-----|
| 業界平均との比較数値 | +20%程度 | 退職率9% vs 20%+ |
| 特定競合との具体的比較 | +10-20% | AZO対比のDIFM補充頻度 |
| ネガティブケースの数値 | +10-20% | 断念案件数、見送り基準 |
| 開示データの検証可能性 | +20%程度 | 会員比率の定期開示 |
| 業界構造との数値的合致 | +30%程度 | フラグメント市場 × 規模データ |
