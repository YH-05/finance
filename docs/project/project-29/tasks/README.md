## 前提条件

以下のシリーズがキャッシュされている必要があります：

| シリーズID | 用途 |
|------------|------|
| DFF, DGS1MO〜DGS30 | 金利・イールドカーブ |
| T10Y3M, T10Y2Y | イールドスプレッド |
| BAMLC0A*, BAMLH0A* | 社債スプレッド |
| DTWEXAFEGS | ドル指数 |

キャッシュ同期コマンド：
```bash
uv run python -m market.fred.scripts.sync_historical --all
```
