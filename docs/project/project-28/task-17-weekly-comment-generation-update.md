## 追加するコメントテンプレート

### 金利コメント（400字）

```markdown
テンプレート構造:
1. 週間サマリー（1文）
2. 主要金利の動向（2-3文）
   - 2年債、10年債、30年債
3. イールドカーブ分析（2文）
   - スプレッドの変化
   - 逆イールドの有無
4. FF金利と政策見通し（1-2文）
5. 市場への影響（1文）

生成時の参照データ:
- interest_rates.data（各シリーズの値と変化量）
- interest_rates.yield_curve（スプレッド、逆イールド判定）

生成例:
今週の米国債市場は、長期金利が上昇する一方、短期金利は横ばいとなった。
10年債利回りは4.50%（前週比+8bp）、2年債利回りは4.25%（前週比+2bp）と、
長短金利差が拡大した。

イールドカーブは引き続き順イールドを維持しており、2年-10年スプレッドは
+25bpと前週から+6bp拡大した。景気後退懸念の後退を示唆している。

FF金利は4.50%で据え置きとなっており、3月FOMCでの利下げ観測は30%程度に
留まっている。金利上昇は株式市場のバリュエーションに若干の重しとなる
可能性がある。
```

### 為替コメント（400字）

```markdown
テンプレート構造:
1. 週間サマリー（1文）
2. ドル円の動向（2-3文）
3. その他主要通貨の動向（2-3文）
4. 為替要因分析（1-2文）
5. 今後の見通し（1文）

生成時の参照データ:
- currencies.symbols（各通貨ペアのレートと変化率）
- currencies.summary（最強/最弱通貨、円のトレンド）

生成例:
今週の為替市場は、米ドルが主要通貨に対して全般的に堅調だった。
ドル円は155.50円（前週比+0.82%）と円安が進行し、年初来では
+2.50%の円安となっている。

ユーロ円は168.20円（前週比+0.45%）、ポンド円は197.80円
（前週比+0.65%）と、いずれも円安方向に推移した。豪ドル円は
102.30円（前週比-0.35%）と、豪ドルの弱さが目立った。

円安の背景には、日米金利差の拡大と、日銀の追加利上げに対する
慎重姿勢がある。今後も日銀政策とFRBの利下げ観測が円相場を
左右する展開が続きそうだ。
```

### 来週の材料コメント（400字、拡張）

```markdown
テンプレート構造:
1. 概要（1文）
2. 注目決算（3-4文）
   - MAG7企業の決算
   - その他主要企業
3. 重要経済指標（2-3文）
   - 雇用統計、CPI等
4. その他イベント（1文）
5. 市場への影響（1-2文）

生成時の参照データ:
- upcoming_events.earnings（決算予定）
- upcoming_events.economic_releases（経済指標発表）
- upcoming_events.summary（ハイライト）

生成例:
来週は決算シーズンのピークと重要経済指標の発表が重なる。

決算面では、2月1日にApple（AAPL）、2月3日にMicrosoft（MSFT）と
Alphabet（GOOGL）の決算発表が予定されている。特にAppleは中国
市場の動向、MicrosoftはAIクラウド事業の成長が焦点となる。
その他、Visa（V）、Mastercard（MA）の決算も消費動向を占う
上で注目される。

経済指標では、2月7日に雇用統計、2月12日に消費者物価指数（CPI）
の発表が控えている。雇用統計では非農業部門雇用者数18万人増が
市場予想となっており、これを大きく上回ると利下げ期待が後退する
可能性がある。

これらの材料が出揃うまで、市場は様子見姿勢を強める展開となりそうだ。
```
