# Weekly report generator improvement plan

- 2026-01-27

## 概要

performance_agent.pyによって株式市場インデックスやセクター動向、個別株、コモディティのパフォーマンスは取得できているが、まだ足りない情報がある。

### 不足情報

- 先物データ(ただしyfinanceからは取得できるデータが限定的と思われる)。
- 金利データ。米国金利はFREDから収集可能。それ以外の国の金利は別のソースから個別に収集する必要がある。
- ニュース記事。データを元に仮設構築→仮設検証のためにニュース取得が不可欠。

## 対処法

- 先物データ: yfinanceから取得可能なデータに限定。
- 金利データ: FREDから米国金利データ(各年限)を収集。米国以外の金利は追加実装するが、ここでは実装しない。
- ニュース記事: Web fetchやGemini search、tavily、RSS mcpサーバーなどで対応可能。ただし、検索クエリの最適化フェーズ(後述が必要)
