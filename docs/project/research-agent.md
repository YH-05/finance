## 背景と目的

### 背景

- 背景の種類: 新しいニーズが発生
- 詳細: 新機能の追加と既存機能の拡張
- 具体的な状況: note記事用に画像を収集するエージェントを追加したい。特定サイトからデータを収集する機能が必要。

### 目的

新しいエージェントを `.claude/agents/` に追加し、note記事作成のための画像収集・リサーチ機能を提供する。
