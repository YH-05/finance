## 共通要因

### 1. トランプ関税25%（1月14日発表）

トランプ大統領は1月14日、先端半導体に対する25%関税を発動する大統領令に署名しました。

- **対象**: NVIDIA H200、AMD MI325Xなど
- **発効日**: 2026年1月15日
- **免除対象**: 米国内データセンター向け、研究開発用、修理・交換用
- **背景**: 9ヶ月間のSection 232調査に基づく国家安全保障上の措置

### 2. AI投資の回収懸念

ビッグテック各社の巨額AI投資に対し、回収時期の不透明感が市場心理を圧迫しています。

| 企業 | 2025-2026年設備投資 |
|------|---------------------|
| Microsoft | 940億ドル以上 |
| Meta | 700-720億ドル（さらに拡大予定） |
| Amazon | 1,250億ドル |
| Alphabet | 数百億ドル規模 |

### 3. 利益成長の鈍化

Mag7の2026年利益成長率は18%予想（2022年以降最低）。S&P500の残り493銘柄の13%成長と大差なくなっています。

### 4. バリュエーション懸念

- **Mag7平均PER**: 29倍
- **S&P500平均PER**: 22倍
- **Nasdaq100平均PER**: 25倍

### 5. 市場ローテーション

2025年11月以降、「Impressive-493」（S&P500からMag7を除いた銘柄群）がMag7をアウトパフォーム。Yardeni Researchは「この傾向は2026年も継続する」と予測しています。

### 1. NVIDIA（エヌビディア）

#### 概要

NVIDIAは2026年1月、米中両国の板挟みとなる厳しい状況に直面しています。最大の懸念材料は中国市場へのアクセス問題です。1月14日、中国当局が税関職員にH200チップの輸入をブロックするよう指示したと報じられ、株価は即座に1%以上下落しました。

#### 詳細分析

この問題の背景には複雑な地政学的事情があります。トランプ政権は2025年12月にH200チップの中国への輸出を条件付きで承認しましたが、その条件として売上の25%を米国政府に納めるという「上納金」を要求しました。これに対し中国政府は反発し、国内テック企業に対してNVIDIA製品の注文を控えるよう指導を開始。Jensen Huang CEOは1月初旬のCESで「中国でのH200需要は依然として強い」と強調しましたが、市場の不安を払拭できていません。

数字で見ると、中国のテック企業はすでに200万個以上のH200チップを発注していますが、NVIDIAの在庫は約70万個に過ぎず、需給ギャップが生じています。NVIDIAはTSMCに増産を依頼し、中国顧客には全額前払いを要求するなど異例の対応を取っています。1台あたり約27,000ドルという価格設定で、中国市場は推定500億ドル規模のAI市場として重要性を持ちます。

さらに、1月14日にはトランプ大統領が先端半導体に25%の関税を課す大統領令に署名。NVIDIAのH200やAMDのMI325Xが対象となりました。米国内データセンター向けなどには免除措置がありますが、市場の反応は冷ややかでした。10月末の史上最高値から9.1%下落し、時価総額は約4,600億ドル消失しています。41人のアナリストのうち39人が依然「買い」を維持していますが、短期的なボラティリティは避けられない状況です。

#### 主要リスク

- 中国H200輸入禁止措置の長期化
- 米中チップ戦争のさらなる激化
- 国内チップメーカー（Huawei、Cambricon）の台頭

#### アナリスト見通し

- コンセンサス: Strong Buy（39 Buy / 1 Hold / 1 Sell）
- 平均目標株価: 現在価格から37%上昇を示唆

### 2. Tesla（テスラ）

#### 概要

Teslaは2026年1月、EV業界の王座から転落するという歴史的な転換点を迎えました。1月2日に発表された2025年通年の販売台数は164万台で、前年比8.6%減少。これは同社史上最大の年間減少幅であり、2年連続のマイナス成長となりました。第4四半期単独では前年同期比15.6%減の約41.8万台と、さらに深刻な落ち込みを記録しています。

#### 詳細分析

この結果、中国BYDがEV販売台数で世界首位の座を奪取しました。BYDは2025年に約226万台のEVを販売し、Teslaを大きく引き離しています。かつてEV市場を切り開いたTeslaが、創業から約20年で首位の座を明け渡すことになりました。

地域別に見ると、欧州での落ち込みが特に深刻です。欧州全体で28%減、ドイツで48%減、フランスで38%減と壊滅的な数字が並びます。この背景には、Elon Musk CEOの政治的言動があると指摘されています。ドイツの極右政党AfDやフランスのマリーヌ・ルペン氏への支持表明が、環境意識の高い欧州の顧客層の反感を買い、「ブランド離れ」を引き起こしているとの分析があります。AP通信は「マスク氏の右傾化への顧客の反乱」と表現しています。

米国市場でも、2025年10月1日に7,500ドルのEV税額控除が廃止されたことが大きな打撃となりました。第3四半期は駆け込み需要で記録的な49.7万台を販売しましたが、第4四半期は反動で急落。さらにトランプ政権がクリーンエア規制の緩和を進めており、逆説的にTeslaのEV専業モデルには逆風となっています。

バリュエーション面では、PERが約200倍とS&P500で2番目に高い水準にあり、これは販売減少が続く企業としては「法外」との批判があります。ただし、投資家の多くはロボタクシーや人型ロボット「Optimus」に期待を寄せており、従来の自動車メーカーとしてではなく、AI・ロボティクス企業として評価しています。Musk氏は2026年1月からUAEでFSDを提供開始し、2〜3月には欧州での完全自動運転承認を見込んでいると発言しています。

#### 販売データ

| 指標 | 数値 |
|------|------|
| 2025年通年販売 | 164万台（-8.6% YoY） |
| Q4 2025販売 | 41.8万台（-15.6% YoY） |
| 欧州販売 | -28% YoY |
| ドイツ | -48% YoY |
| フランス | -38% YoY |
| 中国販売 | 約60万台（全体の30%） |

#### 主要リスク

- BYD、Xiaomi、XPengとの中国競争激化
- 欧州でのブランドイメージ悪化
- 高バリュエーション（PER約200倍）

#### ポジティブ要因

- ロボタクシー事業への期待
- FSD（完全自動運転）の海外展開
- 人型ロボット「Optimus」

### 3. Apple（アップル）

#### 概要

Appleは2026年1月、AI戦略における重大な転換点を迎えました。1月12日、GoogleのGemini AIを次世代Siriおよび Apple Intelligence機能の基盤として採用する複数年契約を発表。これは事実上、AI開発競争において自社単独での勝負を断念し、競合であるGoogleに依存する「AI降伏」と一部で評されています。

#### 詳細分析

この決定の背景には、AppleがOpenAI、Google、Anthropicといったスタンドアロンの生成AIプレイヤーに対し、技術的に遅れを取っているという現実があります。アナリストのMing-Chi Kuo氏によると、AppleはAI専用サーバーチップの量産を2026年後半に開始する予定ですが、それまでの「時間稼ぎ」としてGoogle提携を選択したとのことです。Wedbushのダニエル・アイブス氏は、この提携を「Googleにとっては主要基盤モデルとしての検証、Appleにとっては2026年以降のAI戦略加速への足がかり」と評価しています。

ただし、この提携には複雑な影響があります。Appleは現在、GoogleからデフォルトのSearch契約として年間約200億ドルを受け取っており、これはほぼ純利益に近い収益源です。AIがクエリ型検索からエージェント型に移行する中、この契約の持続可能性に疑問が生じています。また、Search契約自体が独占禁止法の審査対象となっており、今回のAI提携がその交渉をさらに複雑にする可能性があります。

中国市場では、2025年後半にiPhone 17が好調で、10〜11月には20%以上のシェアで首位を獲得。通年でもHuaweiに僅差の2位（共に約17%）となりました。しかし、2026年については基本モデルのiPhone発売が2027年初頭に延期されたことから、出荷台数が4.2%減少するとIDCは予測しています。

一方でTSMCとの関係にも変化が見られます。AI需要の急増により、NVIDIAやAMDがTSMCの先端プロセス（N2など）の生産枠を奪い合う状況となり、Appleがかつて享受していた「最優先顧客」としての地位と価格優位性が薄れつつあります。次世代iPhone向けA20チップの製造コスト上昇が報じられています。

#### Google提携の影響

| 項目 | 内容 |
|------|------|
| 提携内容 | GeminiがApple Intelligence/Siriの基盤に |
| メリット | AI機能の迅速な展開、開発コスト削減 |
| リスク | 競合依存、Search契約への影響 |
| 年間Search契約 | 約200億ドル（ほぼ純利益） |

#### 主要リスク

- AI競争での技術的劣後
- Google依存による戦略的制約
- TSMCでの優先度低下
- 中国市場での競争激化（Huawei、Xiaomi）

### 4. Microsoft（マイクロソフト）

#### 概要

Microsoftは2026年1月、AI投資の規模と回収時期を巡る投資家の懸念に直面しています。2026年度第1四半期の設備投資は約350億ドルに達し、通年では940億ドル以上と見込まれています。これは前年の約650億ドルから45%増という驚異的なペースです。

#### 詳細分析

CFOのAmy Hood氏は当初「2026年度は設備投資の伸びが鈍化する」と述べていましたが、その後予想を撤回し「加速する」と修正。この方針転換が市場の不安を招き、株価は2.9%下落しました。

収益面では、Microsoft Cloudの年間売上が750億ドルを突破し、Azure収益が急速に成長しています。しかし、クラウド粗利益率は68%に低下し、第2四半期は66%程度に落ち込む見通しです。AI機能の拡張とインフラ構築のコストが利益率を圧迫しています。さらに、OpenAIへの投資から31億ドルの損失を計上したことも純利益に影響を与えました。

クラウド市場での競争も激化しています。AWSが約29〜30%のシェアでリードを維持する中、Azure は約20%で2位。Google Cloudが13%まで急成長しており、OracleもGartnerのMagic Quadrantで「リーダー」に選出されるなど、競争環境は厳しさを増しています。

一方で、AI PCの販売は期待を下回っています。TipRanksの報道によると、消費者がAI機能に「困惑している」状態で、需要喚起に苦戦しています。ただし、Copilotの企業向け導入は順調で、Microsoft 365やDynamics 365のサブスクリプション収益が成長を支えています。

アナリストの見方は依然として強気で、55人中52人が「買い」または「強い買い」を維持。ウォール街のコンセンサスでは、2026年の株価上昇幅がトリリオンダラー企業の中で最も大きいと予想されています。

| Cloud粗利益率 | 68%（Q2は66%予想） |
| OpenAI投資損失 | 31億ドル |

#### クラウド市場シェア

- 2026年売上成長予想: 16%
- トリリオンダラー企業中、最大の上昇余地

### 5. Meta（メタ）

#### 概要

Metaは2026年1月、戦略の大転換と大規模リストラの渦中にあります。1月12日、Reality Labs部門の約10%にあたる1,500人以上の人員削減を発表。VRゲームスタジオ3社を閉鎖し、「メタバースからAIウェアラブルへの投資シフト」を鮮明にしました。

#### 詳細分析

Mark Zuckerberg CEOがかつて「会社の未来」と呼んだメタバースからの撤退が鮮明になっています。Reality Labsは過去6年間で700億ドル以上の損失を計上しており、直近四半期だけでも44億ドルの赤字（売上はわずか4.7億ドル）。2026年のReality Labs売上は前年同期比で減少する見通しとCFOのSusan Li氏が明言しています。

閉鎖されたスタジオ:
- **Sanzaru Games**: Asgard's Wrath開発
- **Twisted Pixel**: Deadpool VR開発
- **Armature Studio**: Resident Evil 4 VR開発

代わりに注力するのがAIインフラと「ウェアラブル」です。1月12日、Zuckerberg氏は「Meta Compute」という新イニシアチブを発表。2020年代中に「数十ギガワット」規模のAIインフラを構築する計画で、2028年までに米国内で6,000億ドル以上の投資を行うと宣言しています。この発表に合わせ、元共和党政府高官でゴールドマン・サックス幹部のDina Powell McCormick氏を社長に起用しました。

ウェアラブル分野では、EssilorLuxottica（Ray-Ban親会社）との提携を強化。Ray-Ban Meta smart glassesは累計200万台以上を販売し、2026年末までに年間生産能力を1,000万台に拡大する計画でしたが、現在は生産能力を2〜3倍に増やす協議を進めています。

2025年10月には、設備投資見通しを700〜720億ドルに3度目の上方修正し、2026年は「さらに大幅に増加」と発表。この発表後、株価は11.33%急落し、1日で2,147億ドルの時価総額が消失しました。これは同社史上2番目に大きな1日の損失でした。

#### Reality Labs財務

| 指標 | 数値 |
|------|------|
| 過去6年間累積損失 | 700億ドル以上 |
| 直近四半期損失 | 44億ドル |
| 直近四半期売上 | 4.7億ドル |
| 人員削減 | 約1,500人（10%） |

#### 戦略転換

| FROM | TO |
|------|-----|
| メタバース・VR | AIウェアラブル |
| VRヘッドセット | スマートグラス |
| VRゲーム開発 | AI機能開発 |

#### 主要リスク

- 巨額AI投資の回収時期不透明
- メタバース撤退による評価損
- 競合（Apple Vision Pro等）との競争

### 6. Amazon（アマゾン）

#### 概要

Amazonは2025年に約5%の株価上昇にとどまり、マグニフィセント7の中で最もパフォーマンスが悪い銘柄となりました。しかし、複数のアナリストは2026年を「Amazonの年」と位置づけ、復活に期待を寄せています。

#### 詳細分析

最大の焦点はAmazon Web Services（AWS）です。AWSは会社全体の営業利益の約60%を生み出す稼ぎ頭ですが、クラウド市場でのポジショニングに不透明感があります。Bank of Americaは「AWSのポジショニングに関する不確実性が株価に反映されているが、2026年にAWS収益成長が加速し、相対的なAI能力が強化されれば改善の可能性がある」と指摘しています。

小売事業では効率化が進んでいます。物流ネットワークの最適化やコスト削減により、利益率は改善傾向にあります。また、広告事業は過去4四半期で約650億ドルの高マージン収益を生み出しており、Amazon.comのトラフィックを収益化する隠れた成長ドライバーとなっています。Prime会員サービス、Alexa・Echoデバイスエコシステムも堅調です。

一方でリスク要因も存在します。最大の懸念はFTC（連邦取引委員会）との反トラスト訴訟で、2026年10月に裁判が予定されています。これが投資家心理を圧迫し、「利益確定売り」を招いているとの見方があります。また、1,250億ドル規模のAI関連設備投資についても、回収時期の不透明さが指摘されています。

バリュエーション面では、PERが約34.2倍と、市場はクラウド事業を中心に評価していますが、EC事業の利益率改善や広告事業の成長は十分に織り込まれていないとの見方もあります。Motley Foolは「2026年初頭において最も有望なマグニフィセント7銘柄」としてAmazonを挙げ、Wolfe Researchも「It's Amazon's Year（2026年はAmazonの年）」と強気のコメントを発表しています。

#### 事業セグメント

| セグメント | 状況 |
|------------|------|
| AWS | 営業利益の約60%、成長加速に期待 |
| 小売 | 効率化進展、利益率改善 |
| 広告 | 過去4Q累計650億ドル、高マージン |
| Prime | 会員基盤安定 |

#### 主要リスク

- FTC反トラスト訴訟（2026年10月）
- AWS競争激化（Google Cloud、Azure）
- 1,250億ドル設備投資の回収

#### ポジティブ要因

- 「2026年はAmazonの年」との複数アナリスト評価
- EC利益率改善
- 広告事業の成長

### 7. Alphabet（アルファベット / Google）

#### 概要

Alphabetは2026年1月、マグニフィセント7の中で最もポジティブな展開を見せている銘柄です。1月12日、時価総額が4兆ドルを突破し、NVIDIA、Apple、Microsoftに続く4社目の「4兆ドルクラブ」入りを果たしました。2025年の株価上昇率は65%と、グループ内でトップクラスのパフォーマンスを記録しています。

#### 詳細分析

最大の追い風は、同日発表されたAppleとの複数年AI提携です。GoogleのGemini AIが次世代Siriおよび Apple Intelligence機能の基盤に採用されることが決定。これにより、Geminiは20億台以上のAppleデバイスにアクセスする道が開かれ、Androidエコシステムを超えた展開が可能になりました。Jim Cramer氏は「Geminiが勝者だ」と宣言し、Alphabetへの「own it, don't trade it（保有し続けろ）」姿勢を再確認しています。

Cantor Fitzgeraldのアナリストは「2025年の強いアウトパフォーマンスの後、株価は中期レンジを上抜けし、短期的な期待は高いが、トップライン成長が加速する限りプレミアムバリュエーションは維持される」と評価。Rosenblattのアナリストは目標株価を業界最高の390ドルに引き上げました。

Google Cloudも重要な成長エンジンとなっています。第3四半期の売上は前年比16%増の1,023億ドル、EPSは35%増の2.87ドルと好調。YouTube、検索、サブスクリプションも二桁成長を維持しています。TipRanksのアナリストは「2026年も積極的な姿勢を維持し、AIをエコシステム全体に浸透させる努力を倍増させる」と予想しています。

さらに、WalmartとのAIショッピング提携も発表されました。Gemini AIを活用したGoogle Payチェックアウト機能が導入され、「アジェンティックコマース（AIエージェントによる商取引）」の先駆けとなっています。

リスク要因としては、DOJ（司法省）による独占禁止法訴訟があり、検索事業の分割などが議論されています。しかし、トランプ政権下では規制環境が緩和されるとの期待もあり、Cantor Fitzgeraldは「規制環境の緩和とGeminiを通じたAI能力が株価をさらに押し上げる」と見ています。
