# market_analysis プロジェクト

## 概要

金融市場のデータ取得・分析・可視化を行う Python ライブラリ。
グローバル市場（株式、為替、債券、コモディティ）のデータを yfinance と FRED API から取得し、
テクニカル分析指標の計算、相関分析、チャート生成を提供する。

**主な用途:**
- 株価・為替・指数データの取得と分析
- マクロ経済指標（金利、GDP、インフレ率等）の追跡
- 複数資産間の相関分析
- 分析結果の可視化とエクスポート

## 主要機能

### データ取得（Data Fetching）
- [ ] yfinance による株価・為替・指数データ取得
- [ ] FRED API による経済指標データ取得
- [ ] コモディティ（金、原油等）データ取得
- [ ] SQLite によるローカルキャッシュ
- [ ] yfinance のティッカー管理

### 分析機能（Analysis）
- [ ] 基本指標: 移動平均、リターン、ボラティリティ
- [ ] 相関分析: 複数銘柄間の相関係数、ベータ値
- [ ] セクター分析: セクター別パフォーマンス比較

### 可視化（Visualization）
- [ ] 価格チャート（ローソク足、ライン）
- [ ] テクニカル指標オーバーレイ
- [ ] 相関ヒートマップ
- [ ] PNG/SVG 形式でのエクスポート

### データ出力（Export）
- [ ] pandas DataFrame
- [ ] JSON/CSV エクスポート
- [ ] SQLite データベース保存
- [ ] AI エージェント向け構造化 JSON

### 運用機能（Operations）
- [ ] 日次バッチ処理対応
- [ ] 定期実行（scheduler）対応

## 技術的考慮事項

### 技術スタック
- **データ取得**: yfinance, fredapi
- **データ処理**: pandas, numpy
- **可視化**: plotly（メイン）, marimo（ノートブック）, matplotlib/seaborn（フォールバック）
- **永続化**: SQLite (sqlite3)
- **スケジューリング**: schedule または APScheduler

### 制約・依存関係
- Python 3.12+
- yfinance の API レート制限に注意
- FRED API キーが必要（環境変数 `FRED_API_KEY`）
- 無料 API のため、リアルタイムデータは非対応

### 対象市場
- **株式**: グローバル主要指数（日経225、S&P500、FTSE等）
- **為替**: 主要通貨ペア（USD/JPY、EUR/USD等）
- **債券**: 米国債金利
- **コモディティ**: 金、原油、天然ガス等
