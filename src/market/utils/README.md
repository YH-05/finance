# market.utils

market パッケージの共通ユーティリティモジュール。

## 概要

market パッケージ内で使用する汎用ヘルパー関数を集約するモジュールです。

**現在の状態**: 空モジュール（ユーティリティ関数の追加先として準備済み）

## 今後追加予定の機能

- 日付範囲のバリデーション
- データフレームの共通変換処理
- シンボルの正規化ユーティリティ
- レート制限ハンドラー
