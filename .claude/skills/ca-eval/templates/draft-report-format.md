| プレースホルダー | 説明 |
|----------------|------|
| `[TICKER]` | ティッカーシンボル |
| `[企業名]` | 企業の正式名称 |
| `[report_source]` | アナリストレポートの出典 |
| `[YYYY-MM-DD]` | 生成日 |
