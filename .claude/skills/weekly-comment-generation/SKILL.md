# Weekly Comment Generation

週次レポートの各セクション向けコメント文を生成するスキルです。

## 目的

このスキルは以下を提供します：

- **セクション別コメント生成**: 指数、MAG7、セクター、金利、為替、マクロ経済等のコメント
- **ハイブリッド方式**: テンプレート構造＋LLM生成の組み合わせ
- **文字数管理**: 各セクションの目標文字数を達成
- **ニュース統合**: 関連ニュースをコメントに組み込み

## いつ使用するか

### プロアクティブ使用

`weekly-report-writer` エージェントの第2フェーズとして、`weekly-data-aggregation` の後に呼び出される。

### 明示的な使用

週次レポート生成ワークフローの一部として呼び出し。

## 文字数目標

| セクション | 目標文字数 | 説明 |
|-----------|-----------|------|
| ハイライト | 300字 | 今週の要点を5-7ポイントで |
| 指数コメント | 750字 | 主要指数の動向と要因分析 |
| MAG7コメント | 1200字 | 個別銘柄の動向とニュース |
| 上位セクターコメント | 600字 | 上位3セクターの分析 |
| 下位セクターコメント | 600字 | 下位3セクターの分析 |
| 金利コメント | 400字 | 米国債金利の動向と市場への影響 |
| 為替コメント | 400字 | ドル円・ドル指数の動向と要因分析 |
| マクロ経済コメント | 600字 | Fed、経済指標の動向 |
| 投資テーマコメント | 450字 | AI、半導体等のテーマ |
| 来週の材料 | 400字 | 決算、経済指標発表、注目イベント |
| **合計** | **5700字以上** | |


```markdown
テンプレート構造:
1. 注目テーマサマリー（1-2文）
2. AI/半導体動向（3-4文）
   - 主要銘柄の動向
   - ニュース要因
   - 今後の展望
3. その他注目テーマ（2-3文）
   - クリーンエネルギー
   - 自動運転
   - サイバーセキュリティ等
4. 投資示唆（1-2文）

生成時の参照データ:
- news.by_category.tech
- news.by_category.ai
- mag7関連ニュース

生成例:
今週の投資テーマでは、AI関連銘柄が引き続き市場の中心的な関心事となった。
生成AIの普及拡大とデータセンター投資の加速が、関連銘柄を押し上げている。

AI半導体関連では、エヌビディア（NVDA）がAI需要の持続性について強気の見通しを
示し、週間+1.90%と堅調だった。マイクロソフト（MSFT）のCopilot採用企業が
拡大しているとの報道も、AI投資テーマへの追い風となった。一方、AI電力需要の
増大に対応するため、テック大手がクリーンエネルギー投資を加速しているとの
報道も注目された。今後のAI関連銘柄の動向は、決算発表でのAI収益化の進捗と、
ガイダンスの内容が鍵を握りそうだ。

その他のテーマでは、自動運転技術でテスラ（TSLA）のFSD進捗が注目を集めた。
サイバーセキュリティ関連は、企業のAI導入に伴うセキュリティ投資拡大期待から
堅調に推移した。クリーンエネルギーはソーラー銘柄が金利上昇の影響でやや軟調だった。

テーマ投資では、短期的な過熱感に注意しつつも、AI関連の構造的な成長トレンドは
中長期的に続く可能性が高いと考えられる。
```

### 来週の材料（400字）

```markdown
テンプレート構造:
1. 週間スケジュール概観（1-2文）
2. 決算発表予定（3-4文）
   - 注目企業と発表日
   - 市場への影響予想
3. 経済指標発表（3-4文）
   - 重要指標と発表日
   - 市場予想と注目点
4. イベント/その他（2文）
   - 中央銀行関連
   - 地政学リスク
5. 投資戦略への示唆（1-2文）

生成時の参照データ:
- 決算カレンダー（外部参照）
- 経済指標カレンダー（外部参照）
- news.by_category.macro

生成例:
来週は決算発表と経済指標が集中し、市場のボラティリティが高まる可能性がある。
主要イベントを確認し、リスク管理に注意したい週となりそうだ。

決算発表では、火曜日にテスラ（TSLA）、水曜日にマイクロソフト（MSFT）と
メタ（META）、木曜日にアップル（AAPL）とアマゾン（AMZN）が予定されている。
特にAI投資の収益化進捗と今後のガイダンスが注目される。テスラについては納車
台数見通しとFSD進捗、メタについてはReality Labs損失とAI広告効果が焦点だ。
決算内容によっては、MAG7銘柄を中心に大きな値動きが予想される。

経済指標では、金曜日の雇用統計（非農業部門雇用者数）が最大の注目イベントだ。
予想は+18万人で、労働市場の堅調さが継続するかがポイント。水曜日のFOMC議事録
公開も、Fed の利下げスタンスを探る上で重要だ。火曜日の消費者信頼感指数、
木曜日のPCEデフレーター（インフレ指標）も注目される。

その他、欧州中央銀行（ECB）の政策決定会合が木曜日に予定されている。中東情勢
の動向も引き続き原油価格と市場心理に影響する可能性がある。

週前半はポジション調整が進み、週後半の決算・指標発表で方向性が決まる展開か。
```

## プロセス

```
Phase 1: データ読み込み
├── aggregated_data.json を読み込み
├── 各セクションの参照データを特定
├── 金利・為替データの確認
└── ニュースを関連セクションにマッピング

Phase 2: コメント生成（セクション順）
├── ハイライト生成
│   └── テンプレート + データ埋め込み
├── 指数コメント生成
│   └── テンプレート + LLM補完
├── MAG7コメント生成
│   └── テンプレート + LLM補完 + ニュース統合
├── セクターコメント生成（上位・下位）
│   └── テンプレート + LLM補完
├── 金利コメント生成（新規）
│   └── テンプレート + LLM補完 + interest_rates データ
├── 為替コメント生成（新規）
│   └── テンプレート + LLM補完 + forex データ
├── マクロ経済コメント生成
│   └── ニュース要約 + LLM補完
├── 投資テーマコメント生成
│   └── ニュース要約 + LLM補完
└── 来週の材料生成（拡張）
    └── カレンダー参照 + 決算詳細 + 整形

Phase 3: 文字数調整
├── 各セクションの文字数を計測
├── 目標に対する過不足を確認（目標: 5700字以上）
├── 必要に応じて調整
└── 合計文字数を検証

Phase 4: 出力
└── comments.json を生成
```

```json
  "warning": "合計文字数が目標に達していません",

  "suggestion": "ニュースデータを追加するか、コメントを拡充してください"
```

```json
  "missing": ["interest_rates", "forex"],

}
```
